===========================
Python Package Index - PyPI
===========================

取得
----

* Python3.4

  - ``python -m ensurepip`` (可能會需要 root 權限)
  - 從 Python3.4 開始， ``venv`` 預設會自帶 pip

* Python2

  - Python2.7 的 pip 在 FreeBSD Ports: ``devel/py27-pip`` 可以取得

簡易使用
--------

::

  pip install --upgrade pip
  pip search pycurl
  pip install pycurl
  pip list
  pip uninstall pycurl
