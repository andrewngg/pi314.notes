============
= Mac OS X =
============

停用 spotlight
--------------

::

  sudo launchctl unload -w /System/Library/LaunchDaemons/com.apple.metadata.mds.plist

MacPorts
--------

-   更新 port tree ::

    port selfupdate

-   搜尋 ::

    port search <package_name>

-   安裝 ::

    port install <package_name>
